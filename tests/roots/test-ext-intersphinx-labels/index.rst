Terminology
===========

.. glossary::

   duplicate
   Duplicate
      Both of these terms refer to the same idea: an item that is identical to another.

   nice
      Good, enjoyable, pleasant.

   Nice
      A city in France.

   t : physics
      A variable used to refer to ``time``.

   T : ISO
      A delimiter symbol used to separate date information from time-of-day information.
